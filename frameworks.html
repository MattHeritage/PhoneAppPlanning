To develop this project a list of technologies will be used, they have been listed below...



Main Technologies

All the base web technologies will be used...

HTML5, CSS, and JavaScript


Ionic is the main technology that will power the app, this will handle all the animations for changing page, and all the main components that will be used.


Along with Ionic, Angular is one of the main frameworks that will be used to handle the routing and state for the app and many other things.


Capacitor is the technology that will be used to access all the native mobile features, like the camera and send push notifications.

Other Technologies


Along with the main set of frameworks and technologies used, other frameworks will be used such as Typescript, and SASS.

Personal experience

I have used the basic web technologies HTML, CSS, and JavaScript quite extensively so I have a decent understanding of how they work and how to best use them.

Other than that with other technologies like Angular, Ionic, SASS I have none or very little prior experience, so, I'm going need to spend time to learn them and become proficient with them.

This planning app has been created with an Ionic CDN to help me get a better understanding of how to use the components.
