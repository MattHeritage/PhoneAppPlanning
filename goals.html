What it is

This is a design for a app called "Save it:Item finder" that will be implemented soon, it’s an app that’s focused around keeping track of where items are placed. This will prove especially useful for items that are not needed that often such as batteries, for people who have bad memory, or people who have a lot of things.


Included features

Below, the main features of the application have been listed below...

  • Able to create new items that are then saved this will contain features such as...
    • Name
    • Set photo (Camera or gallery selection)
    • Tags
    • Description
    • Location from map
    • Push notification reminder
  • Existing items can be edited and deleted
  • Able to decide what items show where on the dashboard, to display certain tags or items
  • Can create push notification reminders on certain dates for items
  • Able to search all items by name, or tags
  • Can create custom tags
  • Able to backup items to a server
