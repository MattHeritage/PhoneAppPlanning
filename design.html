To visualise how the app will look and work a set of design images have been created.

This design details what the "add new item" screen will look like, this page contains all the entries for the item information, the user will also see the screen when editing an item and it will autofill the information in.


Press to enlarge

This design shows the dashboard screen, the top display shows the most recent items, this includes items that have been recently viewed, created, or edited.

The following section shows an area where the user can display tags that they have created. The final area shows the upcoming reminders that the users have created this will show the reminders that are due within a month.
At the very bottom of the screen is the tabs that will allow the user to switch from the "dashboard" view to the "Items" view


Press to enlarge

This design shows the group or tag page. There is an image at the top of the screen as the cover image, then below that the items in that group/tag are listed. In this image the items are displayed in grid view, there is a toggle to change them in to list view as well as filter options to help find an item.


Press to enlarge

This design shows the ‘Items’ page this is where all the items stored in the application are shown, various filters can be applied to help find items such as by tag, date created, name. The top right has a search bar, and a button to toggle view type from list to grid view. Items can be selected by tapping them.


Press to enlarge

The final design shows the ‘History’ screen, this is where a list of all the changes are stored. This is useful to keep track of where items where at certain dates or when new items are added. The search bar can be used to filter.
